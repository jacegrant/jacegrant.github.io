Football Tickets

A way for students to manage their tickets

Auburn University Football Student Ticketing System (AUFSTC) is a web application used by students to request tickets, give their tickets away, request guest passes, manage their penalty points, and a lot more regarding their football tickets. With a lot of information on the site, the Atheletic department would always get repeated questions so they came to Campus Web Solutions to add a FAQ page to streamline these questions and answers.

I utilized HTML, CSS, and Javascript to create this page. The FAQ's are held in BootStrap's accordians and can be opened/closed on click.

What you are seeing is the non-admin view of the FAQ's. The admin view has the ability to edit these questions on the page and then that edited information is sent back to the controller through Javascript and ajax, then put into the database tabled called tblQuestion using C#, SQL, and Entity Framework.
