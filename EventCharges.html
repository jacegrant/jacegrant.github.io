Event Charges

A page for admin to charge users

This page allows for admin to apply ticket/parking pass charges for users. Once the admin has picked a user to view, the data table is populated with the tickets and parking passes assigned to that user and they have the ability to add any of those to the cart. Once added, they are viewable in the slide out cart where you can see the overall prices and confirm the charges or send those prices to touchnet if they had not paid yet.

Again I made the view is made with HTML, CSS, Bootstrap with Javascript and jQuery controlling most of the cart and datatable features. The tickets and parking passes are sent to the controller using Javascript, jQuery, and Ajax and then handled with C#, Linq, SQL, and Entity Framework to log the charges and place the transactions in the database. The Touchnet API is used to help assign the charges to the correct user and the correct price.
