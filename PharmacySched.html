Pharmacy Scheduler

A website made for scheduling vacccines/tests

During the early stages of the Covid pandemic, Campus Web Solutions was asked to create a web application for students and faculty members to schedule Covid testing appointments when needed. Covid vaccines, Flu testing and vaccines, and many other test/vaccines would later be implemented. This applicatin would then be opened to the public which increased traffic tremendously. To help with speeds, we transformed lengthy C# code into SQL stored procedures and refactored code to maintain adequate wait and load times.

I utilized HTML, CSS, and Bootstrap for the view of this page and Javascript, jQuery, Ajax, C#, Linq, and Entity Framework for the managing of appointments and getting that information to and from the database. To avoid users from scheduling the same appointments, I implemented database transactions in which whoever sent their appointment request first, there would be a lock on all database tables involved and that initial appointment would be created for that first request, and if there was another user that requested the same appointment, they would see a small message indicating that the appointment selected was just filled and asked to select another time slot.
